## RNTutorial

### 代码怎么来的

目前是`ReactNative 0.59`版本

```sh
npm install -g react-native-cli@^1.0.0
react-native init RNTutorial --version 0.59.5
```

### 如何运行

```sh
git clone git@github.com:pzxbc/RNTutorial.git
cd RNTutorial
yarn install
# IOS平台
react-native run-ios
# Android
react-native run-android
```
